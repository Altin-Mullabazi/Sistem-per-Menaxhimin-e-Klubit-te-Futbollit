# Frontend (Minimal Scope)

## Cfare perfshin

- Login flow
- Players CRUD UI
- State Management me Context API
- Axios client me token interceptor
- Auto refresh token ne 401
- Token vetem ne memory dhe ne header

## Environment

- .env.development
- .env.production

Variabla kryesore:

```env
VITE_API_URL=http://localhost:5000/api
```

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
