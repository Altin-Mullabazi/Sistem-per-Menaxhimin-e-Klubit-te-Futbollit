# CRUD Form - .NET Core Web API + ReactJS

Ky projekt implementon vetem kerkesat e prompt-it:

- CRUD per Players
- JWT Access Token + Refresh Token
- Refresh Token i ruajtur ne databaze (MSSQL)
- React State Management (Context API)
- Axios per API calls
- Bearer token vetem ne Authorization header
- Pa localStorage dhe pa sessionStorage per token
- .env per development dhe production

## Struktura

- BackendAPI: .NET 8 Web API
- Frontend: React + TypeScript + Vite

## Run

1. Backend
- Hape folderin BackendAPI
- Ekzekuto: `dotnet run`

2. Frontend
- Hape folderin Frontend
- Ekzekuto: `npm install`
- Ekzekuto: `npm run dev`

## Build Validation

- Backend: `dotnet build`
- Frontend: `npm run build`
