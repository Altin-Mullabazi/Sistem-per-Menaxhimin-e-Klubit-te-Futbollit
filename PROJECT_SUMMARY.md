# Project Summary

## Scope i Implementuar

Ky projekt eshte implementuar sipas prompt-it dhe perfshin vetem kerkesat kryesore:

- CRUD per Players me .NET Core Web API
- JWT Access Token dhe Refresh Token
- Refresh Token i ruajtur ne databaze MSSQL
- React State Management me Context API
- Axios per komunikim me API
- Bearer token vetem ne header
- Pa ruajtje tokenash ne localStorage/sessionStorage
- .env.development dhe .env.production

## Tech Stack

- Backend: ASP.NET Core 8, EF Core, SQL Server
- Frontend: React 18, TypeScript, Vite, Axios

- Frontend build: kalon
